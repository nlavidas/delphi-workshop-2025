WHAT IS THIS?

A simple, no-authentication local tool to edit your website content.

HOW TO USE:

1. Open the file "local-admin.html" in your web browser

2. Edit your content:

5. Replace the old files in the "content" folder with these new files:
   - Move downloaded files to: c:\Users\nlavi\CascadeProjects\personal-website\content\

6. Commit and push to GitHub:
   Open terminal in the project folder and run:
   
   git add content/
   git commit -m "Update website content"
   git push origin main

7. Wait 1-2 minutes for GitHub Actions to deploy

8. Your changes will be live!
